The Idea

Most payment demos make you
pre-build every product in Stripe.

I wanted a gateway where a customer just types in any amount and pays — instantly. No dashboard setup, no hardcoded Price IDs.

The solution
The Express backend generates the price on the fly using Stripe's inline price_data. One endpoint, any amount, fully dynamic.

What it does

Key features

Dynamic amounts
Any custom USD value, generated server-side with inline price_data — no products pre-configured.
Secure key injection
The Stripe publishable key is served at runtime, never committed to source.
Environment-agnostic
Redirects resolve from request headers — one build runs on localhost, staging or production.
Dockerized & cloud-deployed
Ships with a Dockerfile and runs live on an AWS EC2 Ubuntu instance.
