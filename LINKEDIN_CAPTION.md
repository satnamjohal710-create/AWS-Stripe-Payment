## Option A — Build-in-public (recommended)

🚀 I just built and deployed a full-stack Stripe payment gateway — live on AWS.

"Satnam.pay" lets a customer pay any custom amount, securely, without me having to pre-configure a single product in the Stripe dashboard. The backend generates the price on the fly using Stripe's inline `price_data`.

What it does:
💸 Accepts dynamic, custom USD amounts typed right into the UI
🔐 Injects the Stripe key at runtime so it never sits in the source code
🎨 Clean glassmorphic checkout UI (indigo → purple, dark mode)
🌐 The same build runs unchanged on localhost, staging, or production
🐳 Dockerized and deployed live on an AWS EC2 Ubuntu instance

Tech stack: Node.js · Express · Stripe Checkout · Docker · AWS EC2
